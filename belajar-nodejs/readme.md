# Aplikasi Belajar Node.js

## Struktur Folder

├───config                      -> Berisi konstanta global (nama aplikasi)
├───app.js                      -> File utama untuk menjalankan program
├───README.md                   -> Dokumentasi proyek
└───fungsi                      -> Berisi fungsi sederhana (tampil salam)

## Cara Menjalankan

1. Buka terminal
2. Jalankan perintah:
node app.js
